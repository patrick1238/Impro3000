Pipeline to preprocess the image. The preprocessing steps are gaussian filtering, log transformation, value shifting and the closing operation.

Parameters:
  • image (ArrayList) - Image, which should be preprocessed
  • sigmaVal (Integer) - Sigma value for the gaussian filter
  • kernelSize (Integer) - Kernel size for the closing operation
