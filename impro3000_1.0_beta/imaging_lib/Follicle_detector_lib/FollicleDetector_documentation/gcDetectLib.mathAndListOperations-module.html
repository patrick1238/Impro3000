calculateAutoIntensityThreshold(xPlot, yPlot)

source code 

The function calculates a box plot for given lists of intensity values. The threshold is based on most significant jumps between adjacent boundary points.

Parameters:
  • xPlot (ArrayList) - List of mean intensity values in x direction
  • yPlot (ArrayList) - List of mean intensity values in y-direction
Returns: Float
Intensity threshold for the edge detection
